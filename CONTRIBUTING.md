# Contributing

Contributions must add source-backed GPT-5.6 workflows, tutorials, demos, integrations, benchmarks, or limitations. Open an issue with the public source and proposed case metadata before a large change.

Use the pull request template, update English first, keep case numbers contiguous, synchronize the curated dataset and all localized READMEs, and run every validation command in `docs/maintenance.md`.

Do not submit launch hype, unsourced claims, reconstructed prompts, adjacent-model examples presented as GPT-5.6 evidence, or private material.
